Tips:
1.CUDA 核心数：决定训练速度
2.显存大小：决定可以训练多大的模型以及训练时的最大 Batch Size，对于较大规模的训练而言尤其敏感。
3.屏蔽GPU：
（1）：os.environ[“CUDA_VISIBLE_DEVICES”] = “-1” # 显卡开关
（2）：cpus = tf.config.list_physical_devices (device_type=‘CPU’)

​ tf.config.set_visible_devices (devices=cpus)
4.TensorFlow 2.1 默认安装（pip install tensorflow）就已经同时支持 CPU 和 GPU
5.CPU和GPU谁快：GPU 和 CPU 的选择与模型类型也有关系，例如如果是卷积神经网络则 gtx1050 可能胜过 i7，但如果是强化学习则不一定。
